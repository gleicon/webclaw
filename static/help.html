🚀 Quick Start

1. Connect Your Accounts

Before using integrations, connect your accounts in Settings → Connected Services

Settings → Connected Services → Click "Connect" on any service

2. Try These Examples

twitter_post text="Hello from WebClaw! 🤖"
gmail_list count=5
calendar_today
github_list_issues state="open"

3. Get Help

help tool="twitter_post" verbose=true

🌐 Web & Search

web_fetch Web

Fetch and extract content from any URL

web_fetch url="https://example.com" max_length=5000
web_search Web

Search the web using multiple search engines

web_search query="latest AI developments" num_results=8

🐦 Social Media (Twitter/X)

Setup Required: Connect Twitter in Settings → Connected Services

twitter_post Social

Post a new tweet to Twitter/X

twitter_post text="Hello from WebClaw! 🤖"
twitter_reply Social

Reply to an existing tweet

twitter_reply tweet_id="123456789" text="Great point!"
twitter_search Social

Search recent tweets (last 7 days)

twitter_search query="#AI news" count=10
twitter_timeline Social

Get recent tweets from your home timeline

twitter_timeline count=20

📧 Email & Calendar (Google)

Setup Required: Connect Google in Settings → Connected Services

Gmail Tools

gmail_send Email

Send an email via Gmail

gmail_send to="friend@example.com" subject="Hello" body="How are you?"
gmail_list Email

List recent emails from inbox

gmail_list count=10 label="INBOX"
gmail_read Email

Read full content of a specific email

gmail_read message_id="abc123"
gmail_search Email

Search emails using Gmail query syntax

gmail_search query="from:boss@company.com subject:urgent" count=5

Calendar Tools

calendar_list Calendar

List upcoming calendar events

calendar_list days=7 count=10
calendar_create Calendar

Create a new calendar event

calendar_create title="Team Meeting" start_time="2024-01-15T14:00:00Z" duration_minutes=60
calendar_delete Calendar

Delete a calendar event by ID

calendar_delete event_id="event_123"
calendar_today Calendar

Get today's calendar events

calendar_today

🔧 Developer Tools (GitHub)

Setup Required: Connect GitHub in Settings → Connected Services

github_list_issues Dev

List GitHub issues (assigned to you or in a repo)

github_list_issues owner="gleicon" repo="webclaw" state="open" count=20
github_list_prs Dev

List open pull requests in a repository

github_list_prs owner="gleicon" repo="webclaw" state="open"
github_create_issue Dev

Create a new GitHub issue

github_create_issue owner="gleicon" repo="webclaw" title="Bug: Login fails" body="Steps to reproduce..."
github_search_code Dev

Search code across GitHub repositories

github_search_code query="repo:gleicon/webclaw TODO language:go"
github_comment Dev

Add a comment to an issue or pull request

github_comment owner="gleicon" repo="webclaw" number=42 body="LGTM! 🚀"

📝 Knowledge Base (Notion)

Setup Required: Connect Notion in Settings → Connected Services

notion_list_databases Knowledge

List available Notion databases

notion_list_databases
notion_query Knowledge

Query a Notion database with optional filters

notion_query database_id="Tasks" filter_property="Status" filter_value="Not Started"
notion_read Knowledge

Read content of a Notion page

notion_read page_id="abc123" include_content=true
notion_update Knowledge

Update a Notion page (properties only)

notion_update page_id="abc123" properties={"Status": "Done"}
notion_search Knowledge

Search pages and databases in Notion

notion_search query="project roadmap" limit=10
