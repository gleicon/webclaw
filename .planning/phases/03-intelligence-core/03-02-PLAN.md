## Goal

Implement the agent loop that runs in a Web Worker for non-blocking operation. Assembles context (system prompt + identity + history), calls provider with streaming, and delivers tokens to the UI via message passing.

## Requirements Addressed

- AGNT-01: Agent executes a turn with context assembly → provider call → response handling
- AGNT-04: Agent loop runs in Web Worker
- PROV-03: Provider supports streaming completions to UI

## Must-Haves (Goal Verification)

1. Main thread UI remains responsive during 30-second LLM response
2. Tokens appear in UI within 100ms of provider delivery
3. Web Worker posts TOKEN and COMPLETE messages to main thread
4. Context assembly includes system prompt + identity files + conversation history
5. Agent can abort an in-progress stream
