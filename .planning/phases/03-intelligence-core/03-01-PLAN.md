## Goal

Implement the LLM provider routing system that parses `vendor/model-id` strings and routes to the correct provider (Anthropic, OpenAI, OpenRouter) using SSE streaming via JS fetch bridge. Includes failover with exponential backoff.

## Requirements Addressed

- PROV-01: Agent routes LLM calls using `vendor/model-id` format
- PROV-02: All provider HTTP calls go through `syscall/js` fetch()
- PROV-04: Provider failover with exponential backoff
- PROV-05: Supported providers: Anthropic, OpenAI, OpenRouter

## Must-Haves (Goal Verification)

1. A request to `anthropic/claude-sonnet-4-5` routes to Anthropic API with correct headers
2. A request to `openai/gpt-4` routes to OpenAI API with correct headers
3. A request to `openrouter/anthropic/claude-sonnet-4-5` routes to OpenRouter
4. 5xx errors trigger retry with exponential backoff (1s, 2s, 4s)
5. All HTTP calls use syscall/js fetch bridge, not net/http
