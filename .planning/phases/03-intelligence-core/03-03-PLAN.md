## Goal

Implement conversation context management with automatic summarization when thresholds are exceeded. When history hits 20 messages or 75% of context window, trigger progressive summarization. Before compaction, flush durable knowledge to MEMORY.md per OpenClaw compatibility.

## Requirements Addressed

- AGNT-02: Context history capped at 20 messages or 75% of context window
- AGNT-03: Summarization performed by LLM, summary replaces history
- MEM-04: Before compaction, durable knowledge flushed to MEMORY.md

## Must-Haves (Goal Verification)

1. Conversation with 25 messages triggers summarization automatically
2. Summary replaces old messages, keeping last 3 turns in full
3. MEMORY.md receives extracted facts before compaction
4. User experiences continuous conversation without interruption
5. Token count estimation uses ~4 chars per token heuristic
